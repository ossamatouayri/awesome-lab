## ü§ù Contributing

We love contributions! Here's how you can help:

### Adding Resources

1. Fork this repository
2. Find the appropriate section (use the navigation links!)
3. Add your resource in alphabetical order using this format:
   ```markdown
   - **[Resource Name](URL)** - Brief, clear description
   ```
4. Submit a pull request

### Guidelines

- ‚úÖ Resources must be free or have substantial free content
- ‚úÖ Include a brief, clear description (one line)
- ‚úÖ Ensure links are working and up-to-date
- ‚úÖ Check for duplicates first
- ‚úÖ Place in the correct category and subcategory
- ‚úÖ Maintain alphabetical order within sections

### Quality Standards

We prioritize resources that are:
- **Well-maintained** and regularly updated
- **Beginner-friendly** or clearly labeled for advanced users
- From **reputable sources** (universities, established platforms, recognized experts)
- **Accessible** and inclusive
- **Free** or with generous free tiers
